# Онлайн версия задавальника МФТИ

**Задавальник МФТИ** - бумажная книжка или ПДФ-файл, в котором прописаны программа обучения и задачи ДЗ по всем предметам направления обучения. Он выпускается каждый семестр и раздается на руки студентам. Задавальник - незаменимый инструмент для физтехов, ведь предметов большое количество и бывает сложно отследить, что нужно знать по каждому из них, каковы виды отчетности и когда они будут проходить.

Но несмотря на то, какой замечательной вещью является задавальник, его можно сделать еще лучше. Данный проект является попыткой расширить его функционал.
