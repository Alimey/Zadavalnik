# Онлайн версия задавальника МФТИ

**Задавальник МФТИ** - бумажная книжка или ПДФ-файл, в котором прописаны программа обучения и задачи ДЗ по всем предметам направления обучения. Он выпускается каждый семестр и раздается на руки студентам. Задавальник - незаменимый инструмент для физтехов, ведь предметов большое количество и бывает сложно отследить, что нужно знать по каждому из них, каковы виды отчетности и когда они будут проходить.

Но несмотря на то, какой замечательной вещью является задавальник, его можно сделать еще лучше. Данный проект является попыткой расширить его функционал.


## Что планируется сделать (основная часть)
- Данный проект предназначен для студентов направления ИВТф (2 курс на момент написания), то есть, будет добавляться функционал именно для их задавальника.
- Создать многостраничный сайт, где каждая страница соответствовала бы конкрутному предмету, указанному в официальном задавальнике.
- Переключаться между страницами можно будет с помощью бокового меню.
- На каждой странице будет описание курса, список задач ДЗ, разделенный как в исходном задавальнике. Сохраняется указание дедлайнов, преподавателей, списков литературы, всех ссылок, присутствовавших в исходном ПДФ-файле.
- Номера задач ДЗ будут представлять из себя кнопки, по нажатию на которые высвечивается условие, а в отдельной вкладке - ответ. 
- Будет реализована система авторизации с помощью физтеховской почты. Авторизованный пользователь сможет помечать задачи решенными.

## Что уже сделано
- Запущен сайт с помощью GitHub Pages, где присутствуют страницы курсов по физике и математическому анализу. 
- Подключена база данных, в которой хранятся фотографии задач.
- Реализовано боковое меню и открытие фотографий с условиями по нажатию на кнопки.
- Есть кнопки для закрузки фотографий с условиями (двойной клик по номеру задачи), а также чтобы пометить задачу решенной
